⚠️ Lỗi Truy Cập Micro!

Trình duyệt không thể truy cập micro của bạn.

💡 Cách xử lý lỗi:
1. Hãy chắc chắn bạn đã nhấp chuột vào trang web này ít nhất một lần để trình duyệt kích hoạt quyền hỏi micro.
2. Bảo mật Chrome/Edge: Bạn KHÔNG THỂ mở trực tiếp file HTML tĩnh từ ổ đĩa dạng file:///.... Trình duyệt sẽ chặn mic vì lý do bảo mật. Hãy khởi chạy ứng dụng thông qua Localhost (ví dụ: extension Live Server trên VS Code hoặc công cụ local server bất kỳ) hoặc đưa trực tiếp vào OBS.
3. Nếu đưa vào OBS Browser Source, hãy chắc chắn tích vào ô "Local file" và kiểm tra xem nguồn OBS đã cấp quyền mic chưa.
